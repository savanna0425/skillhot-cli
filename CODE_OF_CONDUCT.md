# Code of Conduct

SkillHot is committed to a welcoming, harassment-free community. Treat participants with respect, assume good faith, and focus criticism on ideas and changes rather than people.

Unacceptable behavior includes harassment, threats, discrimination, sexualized language or imagery, doxxing, and sustained disruption. Maintainers may remove, edit, or reject contributions and may temporarily or permanently restrict participation when this code is violated.

Report conduct concerns privately through [GitHub's security advisory form](https://github.com/savanna0425/skillhot-cli/security/advisories/new). Reports will be handled as confidentially as practical and without retaliation.
